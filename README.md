# Gemini 利用規約チェッカー
利用規約のURLを入力すると、Google Gemini APIを使用して「危険な条項」や「注意すべき点」を自動で分析・要約するWebアプリケーションです。
本来はswallow-7bを利用規約のデータセットでチューニングしたモデルを使用するつもりでしたが、成果発表までにチューニングが間に合わなかったので、gemini-2.5-flashを使用しました。
元のコードはAPIキーを使用していますが、git hubにコードを上げるにあたり、APIキーを入力しなくても問題無いようにデモモードで動くようにしました。
処理部分の大元のpythonのコードは自分が作成しましたが、webで表示するためのhtml,css,flaskの部分はgeminiを使用して調整、出力させました。



## デモモードについて
このリポジトリのコードは、APIキーが設定されていない場合、自動的に「デモモード」として動作します。デモモードでは実際のAPI通信を行わず、あらかじめ用意されたサンプルデータを返します。

## 動作環境
* Python 3.11.5
* Flask

## セットアップ手順（ローカルで動かす場合）

1. リポジトリをクローン
2. 依存ライブラリのインストール
   pip install -r backend/requirements.txt
3. backend/.env.exampleをコピーして、名前を.envに変更します
4. .envファイル内のGEMINI_API_KEYを自身のAPIキーに変更してください。APIキーを入力していない場合、デモモードが起動します
5. backend/app.pyを実行し、frontend/index.htmlを起動してください
